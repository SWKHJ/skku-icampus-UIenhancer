# Campus Cat Mate – Privacy Policy
_Last updated: 2025-11-27_

Campus Cat Mate는 성균관대학교 iCampus(Canvas LMS)의  
대시보드·캘린더 UI 개선 기능과 공부 타이머, 고양이 마스코트 기능을 제공하는 Chrome 확장 프로그램입니다.

본 확장 프로그램은 사용자의 개인 정보를 외부로 전송하거나 제3자에게 제공하지 않습니다.  
모든 데이터는 사용자의 브라우저 내부(Local Storage)에만 저장됩니다.

---

## 1. 서비스 관계 및 접근 범위
Campus Cat Mate는 **iCampus(Canvas LMS)와 공식적으로 연동된 제품이 아닙니다.**  
브라우저 내에서 독립적으로 동작하는 사용자 보조 도구이며,  
iCampus의 서버·계정·제출 시스템에 접근하거나 개입하지 않습니다.

확장 프로그램은 iCampus 관련 페이지에서  
**DOM 요소 확인, UI 개선, 제출 요청 감지(읽기 전용)** 만 수행합니다.  
로그인 정보, 비밀번호, 개인정보 입력란 등 민감 데이터에는 접근하지 않습니다.

---

## 2. 수집하는 데이터
저장되는 데이터는 다음에 한정됩니다:

### • 공부 타이머 기록  
진행 여부, 시작/종료 시각, 누적 시간, 사용자 입력 작업명 등

### • 고양이 마스코트 설정  
색상/밝기/채도, 보유 아이템, 장착 정보 등

### • 포인트 및 상점 정보  
획득 포인트, 구매한 아이템 상태

### • 기능 유지에 필요한 내부 상태  
접속 보상 기록, 디버그 플래그 등

모든 데이터는 **chrome.storage.local**에 저장되며  
**외부 서버로 전송되지 않습니다.**

---

## 3. 데이터 사용 목적
저장된 데이터는 다음 기능 제공의 목적만을 위해 사용됩니다:

- 공부 타이머 기록 유지 및 CSV 다운로드
- 고양이 마스코트 커스터마이징 유지
- 포인트 시스템 및 상점 기능 동작
- UI 개선 기능의 유지 및 안정화

---

## 4. 데이터 외부 전송 여부
Campus Cat Mate는 사용자 데이터를  
**어떠한 외부 서버에도 전송하지 않습니다.**

- Google Analytics ×  
- 광고/트래킹 SDK ×  
- 서버 로깅 ×  
- 외부 API 호출 ×  

모든 기능은 사용자의 브라우저 내부에서만 동작합니다.

---

## 5. 제출·평가 기능 관련 안내
확장 프로그램에는 과제 제출 등의 네트워크 요청을  
**읽기 전용으로 감지하여 애니메이션 재생 및 포인트를 지급하는 기능이 있습니다.**

이 기능은 제출 기능을 **변경, 우회, 개입하지 않습니다.**

---

## 6. 제3자 제공
데이터는 어떠한 경우에도  
외부 기관·업체·제3자에게 제공되지 않습니다.

---

## 7. 데이터 보관 및 삭제
데이터는 Chrome 브라우저 내부에 보관되며 다음의 경우 삭제됩니다:

- 사용자가 확장 프로그램을 제거한 경우
- Chrome 설정에서 “사이트 및 확장 프로그램 데이터 삭제”를 수행한 경우
- 사용자가 chrome.storage.local을 직접 삭제한 경우

개발자는 사용자 장치에 저장된 데이터에 접근할 수 없습니다.

---

## 8. 아동(13세 미만) 보호
본 확장 프로그램은 13세 미만을 대상으로 하지 않으며  
아동 데이터를 수집하지 않습니다.
