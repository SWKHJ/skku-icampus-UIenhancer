# Campus Cat Mate – Privacy Policy
_Last updated: 2025-11-15_

Campus Cat Mate는 성균관대학교 iCampus(캔버스)의 대시보드·캘린더 UI 개선 기능과  
공부 타이머 및 고양이 마스코트 기능을 제공하는 Chrome 확장 프로그램입니다.

본 확장 프로그램은 사용자의 개인 정보를 외부로 전송하거나 제3자에게 제공하지 않습니다.  
모든 데이터는 사용자의 브라우저 내부(Local Storage)에만 저장됩니다.

---

## 1. 수집하는 데이터
Campus Cat Mate는 다음과 같은 정보만 사용자의 장치 내부(Local Storage)에 저장합니다:

- **공부 타이머 상태 및 기록**  
  (진행 여부, 시작 시간, 누적 시간, 사용자 입력 작업명 등)

- **고양이 마스코트 설정**  
  (색상·밝기·채도 등의 조정값, 보유한 아이템 목록, 적용된 커스터마이징)

- **포인트 및 상점 구매 정보**  
  (획득 포인트 수, 구매한 아이템 상태)

- **기능 유지에 필요한 내부 상태**  
  (접속 보상 기록, 디버그 플래그 등)

이 데이터는 모두 **chrome.storage.local에 저장되며 외부로 전송되지 않습니다**.

---

## 2. 데이터 사용 목적
저장된 데이터는 다음 기능 제공을 위해 사용됩니다:

- 공부 타이머의 정상적인 동작 유지  
- 타이머 기록 확인 및 CSV 다운로드  
- 고양이 마스코트의 시각적 상태 유지  
- 포인트 시스템 및 상점 기능 제공  
- 확장 프로그램 기능 보정을 위한 내부 상태 유지  

Campus Cat Mate는 iCampus 화면(UI)을 개선하는 기능을 포함하지만,  
이와 관련된 사용자 설정을 저장하지는 않습니다.

---

## 3. 데이터 외부 전송 여부
Campus Cat Mate는 **모든 사용자 데이터를 외부 서버로 전송하지 않습니다.**

- Google Analytics: 사용하지 않음  
- 광고/트래킹 SDK: 포함하지 않음  
- 원격 서버 로그: 없음  
- 제3자 API 통신: 없음  

확장 프로그램의 모든 기능은 브라우저 내부에서만 실행됩니다.

---

## 4. 제3자 제공
저장된 데이터는 어떠한 경우에도 외부 기관, 제3자, 기타 서비스 제공자에게 전달되지 않습니다.

---

## 5. 데이터 보관 및 삭제
사용자의 모든 데이터는 Chrome 브라우저 내부에 보관되며,
다음의 경우 즉시 삭제됩니다:

- 사용자가 확장 프로그램을 제거한 경우  
- Chrome의 “사이트 및 확장 프로그램 데이터 삭제” 기능을 사용한 경우  
- chrome.storage.local 데이터를 사용자가 수동 삭제한 경우  

개발자는 사용자 장치 내부 데이터에 접근할 수 있는 기술적 수단을 가지고 있지 않습니다.

---

## 6. 아동(13세 미만) 보호
본 확장 프로그램은 교육용 LMS(iCampus) 사용자를 대상으로 하며  
13세 미만 아동을 대상으로 하거나 아동 데이터를 수집하지 않습니다.
